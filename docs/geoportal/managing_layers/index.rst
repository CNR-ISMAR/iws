Managing Layers
===============

| Layers are published resources representing raster o vector spatial data sources. Layers can also be associated with metadata, ratings, and comments.
| In this section, you will learn
- how to find the layers of your interest among all the published layers;
- how to create a new layer by uploading a local data set;
- how to compile metadata and layer info;
- how change the style of the layer;
